# Password Strength Analyzer

This project checks the strength of a password based on security rules.

## 🔍 Features
- Checks password length
- Detects uppercase and lowercase letters
- Checks numbers and special characters
- Classifies password as Weak, Medium, or Strong

## 💻 Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 How to Use
1. Enter a password
2. Click "Check Strength"
3. View the result

## 📌 Purpose
This project helps users create secure passwords and understand basic cybersecurity concepts.
